What if every empty seat was your next ride?

Start Now for Free!

In order to fight the challenges of pollution, traffic, and population increase, cities will need better transportation options. To solve this we’ve implemented a convenient transportation app that connects drivers and passengers that are going the same direction via pre-arranged or on-the-fly means. Our solution provides the ease of ride-hailing apps while being as efficient as public transportation.

Technologies

Ride Surfer is a mobile application, written in TypeScript using the React Native framework. It communicates with a Node.js backend and uses a PostgreSQL database as a backing store.
